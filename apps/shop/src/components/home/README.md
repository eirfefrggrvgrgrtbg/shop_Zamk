# HeroSection Component

## Описание
Минималистичный Hero Section для главной страницы платформы ZAMK с типографическим знаком вопроса.

## Особенности

### Дизайн
- **Строгий минимализм**: использует только белый (#FFFFFF), черный (#000000) и светло-серый (#F9F9F9)
- **Премиальные отступы**: большое количество whitespace для кураторского эффекта
- **Поддержка темной темы**: автоматическая адаптация к dark mode

### Типографика
- **Заголовок**: Playfair Display (serif) с высоким контрастом
- **Кнопка**: Inter (sans-serif) для чистоты интерфейса
- **Размер заголовка**: адаптивный от 2.8rem до 6rem

### Анимация
- **Левитация знака вопроса**: плавные волнообразные движения
- **Интерактивность**: элементы увеличиваются при наведении
- **Плавные переходы**: duration от 8 до 10 секунд для разных элементов
- **Эффект появления**: fade-in с масштабированием

### Типографический знак вопроса
- Составлен из 27 слов "ЗАМК" разного размера и наклона
- Формирует визуальную форму знака вопроса
- Каждый элемент анимируется независимо
- Hover эффекты для индивидуальных слов

### Адаптивность
- **Desktop (lg+)**: полная версия с знаком вопроса
- **Mobile/Tablet**: только левая часть с заголовком и кнопкой
- **Responsive typography**: clamp() для плавного масштабирования

## Технические детали

### Зависимости
- `react-router-dom`: для навигации
- `framer-motion`: для анимаций

### Анимация левитации
- **Y-ось**: от 0 до -6px
- **X-ось**: синусоидальное движение до ±2px
- **Opacity**: колебания от 0.75 до 0.88
- **Scale**: микро-изменения от 1 до 1.01

### Цветовая схема

#### Светлая тема
- Фон: `#FFFFFF`
- Текст: `#000000`
- Градиент фона: `#F9F9F9`

#### Темная тема
- Фон: `#0A0B0D`
- Текст: `#FFFFFF`
- Градиент фона: `#111214`

## Производительность

- Использует CSS transitions для плавности
- Framer Motion оптимизирует анимации через GPU
- Знак вопроса скрыт на мобильных для лучшей производительности
